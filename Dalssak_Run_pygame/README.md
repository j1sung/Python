# Title: Dalssak_Run pygame!
## Game Description
Genre: Arcade

Platform: Pygame
요약: 파이썬 코드로 개발한 간단한 장애물 피하기 게임입니다!
목표: 주어진 시간동안 장애물을 모두 피해 산타 달싹이가 임무를 완수할 수 있도록 도와주세요! 
## How Play?
### 0. 라이브러리 설치 
-> `pip install pygame`
### 1. 모듈화 코드 ver.
`main.py 실행!`
### 2. 전체 통합 코드 ver.
`Dalssak_Run.py 실행!`
